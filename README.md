# Rust Todo

Todo application using rust and axum.

## 🌎 API Spec

|  **name**  | **method** |  **endpoint**  |
|:----------:|:----------:|:--------------:|
| Todo 목록 조회 |    GET     | /api/todo/list |
|  Todo 생성   |    POST    | /api/todo/new  |
|  Todo 수정   |   PATCH    | /api/todo/:id  |
|  Todo 삭제   |   DELETE   | /api/todo/:id  |
